# LEDFlasher

This repository is intended to hold all files related to the LED Flasher used in the oscilloscope lab, as well as instructions for creating building the device. The LED flasher is a simple circuit designed to flash an alternating pair of LEDs on and off with a BNC output to be read by an oscilloscope. It needs a center positive barrel jack power supply outputting between 5VDC and 24VDC.
